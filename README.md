# AutoEpinephrineAlerter-Injector
Automatic injection of epinephrine dose and alerting system for life-threatening allergic reactions
•	Developing a system to automatically detect early symptoms of anaphylaxis using sensor-based monitoring.
•	Designing a mechanism for automatic administration of epinephrine during severe allergic reactions.
•	Implementing a real-time alert system to notify caregivers or emergency contacts.
•	Applying embedded systems and programming concepts to improve emergency response time.
